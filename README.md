# 📊 Cross-Market Digital Ad Performance Dashboard

## 📌 Project Overview
This project presents an end-to-end data engineering and analytics pipeline designed to evaluate digital advertising performance across multiple international markets. The goal is to synthesize raw campaign data into actionable business insights, focusing on key performance indicators (KPIs) such as Click-Through Rate (CTR), Cost Per Click (CPC), and Return on Ad Spend (ROAS).

By analyzing ad formats and demographic engagement, this project provides data-driven recommendations to optimize ad spend allocation and maximize conversion rates.
